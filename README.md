# 📦 WeCare Inventory Management System

A Python-based inventory and invoice management system developed for **WeCare**, a skincare wholesale business. This application helps manage inventory efficiently by handling product sales, restocking, stock updates, and invoice generation through a simple menu-driven interface.

---

## 📖 Overview

The **WeCare Inventory Management System** was developed as part of an academic coursework project using Python. It demonstrates fundamental programming concepts such as modular programming, file handling, dictionaries, exception handling, and user interaction.

The application allows users to:

- View available products
- Sell products
- Restock inventory
- Generate invoices
- Automatically update stock records
- Apply the **Buy 3 Get 1 Free** offer

---

## ✨ Features

- 📋 Display all available products
- 🛒 Sell products with automatic stock updates
- 📦 Restock inventory
- 🧾 Generate sales invoices
- 📝 Generate restocking invoices
- 🎁 Apply the **Buy 3 Get 1 Free** policy
- 🔄 Handle multiple products in one transaction
- ⚠️ Input validation and exception handling
- 📁 Modular code structure
